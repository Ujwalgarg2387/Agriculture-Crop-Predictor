# Agriculture-Crop-Predictor
-This is a Machine Learning based predictive model developed to suggest most suitable crop to grow in the given climatic and soil conditions such as levels of Nitrogen, Phosphorus, Rainfall, pH, Humidity, Temperature, etc.
-Developed and fine-tuned the model with different ML techniques like Logistic regression, K-means clustering, Decision trees, Random forest, XGBoost, etc. to achieve high accuracy rate (98%).
